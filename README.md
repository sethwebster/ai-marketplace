# ai-marketplace

A Claude Code plugin marketplace.

## Install

```bash
/plugin marketplace add sethwebster/ai-marketplace
```

## Install a plugin

```bash
/plugin install <plugin-name>@ai-marketplace
```

## Update

```bash
/plugin marketplace update ai-marketplace
```
